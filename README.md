
## EXASystems Task

## How to install
First, create a database called exasystems_task.

```shell
cp .env.example .env
```
```shell
composer install
```
```shell
php artisan key:generate
php artisan migrate:fresh --seed
php artisan serve
```
```shell
npm install
npm run dev
npm run watch
```
Now enjoy.
